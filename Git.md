# GIT
## Что такое Git ?

* Git - это распределённая система управления версиями которая позволяет отслеживать изменения в файлах, управлять историей проекта и эффективно работать в команде. Это инструмент , который хранит историю изменений на твоем компьютере.

## Зачем нужен Git ?

- Отслеживать изменения в коде.
- Позволяет вернуться к предыдущим версиям.
- Упрощает совместную работу.
- Работает вместе с Github.

## Что такое репозиторий ?

* Репозиторий - это цифровое хранилище данных, предназначенное для сохранения, управления и отслеживания изменений файлов и кода.

## Что такое Github ?

* GitHub - крупнейший веб-сервис для хостинга IT-проектов и их совместной разработки. Можно сказать это социальная сеть для разработчиков. В Github можно хранить свои репозитории в интернете и работать над проектами вместе с другими людьми.

## 💡 Основные команды в Git:
* ' git init ' - Инициализировать репозиторий. > Важно: git init выполняется только один раз
* ' git add имя файла ' - Добавить файлы для следующего коммита. Если написать ' git add . ' то добавяться вообще все файлы.
* ' git commit -m "first commit" ' - Зафиксировать изменения.
* ' git push ' - отправить коммиты в репозиторий.
* ' git config --list ' - лист всех выполненых команд в терминале
